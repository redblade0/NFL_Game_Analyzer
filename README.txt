Uses data from ESPN API: https://site.api.espn.com/apis/site/v2/sports/football/nfl/scoreboard

User will be prompted on CLI which game they want to view updates from.
They will be able to choose which game they want to view by typing in the number correlated with it.
Upon typing the number the score along with the quarter the game is in will be displayed.
The user will be be prompted to quit by pressing q or can type a number to view a different game.
