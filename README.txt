1. Takes in an espn JSON file
2. Updates command line with games that are live
3. User can type which game they want to be displayed
4. HTML File is updated with the game along with some stats

Example of CLI when games visible:

Type the number associated with the game you want to view:

1. New York Giants vs Washington Commanders

Part 3:

Upon typing the game they want to be generated, an HTML file will be generated.

Part 4:

The HTML file will contain the game that they type into.
The two teams logos will be centered in the middle of the page.
The stats for both teams will be shown:
1. QB stats
2. RB stats
3. WR stats
